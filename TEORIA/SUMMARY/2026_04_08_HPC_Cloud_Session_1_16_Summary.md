## 2. Key Concepts

### 2.1 What is Cloud Computing?

- Relatively new area in computer science
- On-demand access to computing resources (servers, storage, databases, networking, software)
- Pay-as-you-go pricing model
- No need for upfront infrastructure investment

### 2.2 Cloud Service Models

| Model | Description | Example |
|-------|-------------|---------|
| **IaaS** (Infrastructure as a Service) | Virtual machines, storage, networks | AWS EC2, Google Compute Engine |
| **PaaS** (Platform as a Service) | Platform for developing/deploying apps without managing infrastructure | **GitHub Pages**, Heroku |
| **SaaS** (Software as a Service) | Ready-to-use software over the internet | Google Docs, Microsoft 365 |

### 2.3 GitHub Pages as PaaS

- **GitHub Pages** is a **PaaS** service
- Hosts static websites directly from a GitHub repository
- Free hosting for personal/organization/project pages
- Supports custom domains (optional)
