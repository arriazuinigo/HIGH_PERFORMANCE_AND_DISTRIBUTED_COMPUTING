## 2. Key Concepts

### 2.1 Cloud Computing in Healthcare

**Opportunities:**
- Save capital expenditure on infrastructure
- Improve patient care and decision-making
- Better coordination among stakeholders
- Patients and healthcare professionals can access/share data more efficiently

**Use Cases:**
| Application | Cloud Provider | Service |
|-------------|---------------|---------|
| **Sequence Analysis** (DNA comparison) | AWS Lambda | Serverless computing via BLAST |
| **Genomic Data Storage** | Google Cloud Platform | GCP genomic storage solutions |
| **Telemedicine** | Microsoft Azure | Telemedicine applications |
| **Remote Patient Monitoring** | IBM Watson Health | IoT + cloud monitoring |

### 2.2 Security and Privacy Regulations

| Regulation | Region | Scope |
|------------|--------|-------|
| **GDPR** | European Union | General data protection |
| **LOPDGDD** | Spain | Spanish data protection |
| **HIPAA** | United States | Healthcare data privacy |

### 2.3 Edge Computing vs Cloud Computing

| Aspect | Edge Computing | Cloud Computing |
|--------|---------------|----------------|
| **Location** | Near data source (devices, laptops) | Centralized servers |
| **Latency** | Low (real-time) | Higher |
| **Processing** | On-device | Remote servers |
| **Use case** | IoT, wearables, mobile health | Large-scale analytics |

### 2.4 GitHub Pages as PaaS (Recap)

- **GitHub Pages = PaaS** (Platform as a Service)
- Hosts static websites from repositories
- Free hosting with custom domain support
- Students must submit their website link on the course main page
