### 1. Introduction to String Operations in Python
- Basic string comparison using the equality operator (`==`).
- Checking if a substring exists in a string using the `in` operator.
- Finding the position of a substring using `.find()` or `.index()` methods.
- These basic methods work for simple patterns but are insufficient for complex pattern matching.

### 2. Regular Expressions (Regex) in Python
- Regular expressions provide a powerful way to extract and match complex patterns in strings.
- Particularly useful for working with log files, configuration files, and data extraction.
- The `re` module in Python provides regex functionality.
